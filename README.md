## User Management System - Microservice Notification

## Installation

```bash
$ npm install
```

## Running the app locally

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running the app docker compose

```bash
$ cd ums.user/app

$ docker-compose up --build 
or 
docker-compose up --build -d
```
